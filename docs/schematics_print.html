DSP Control – Schemi di collegamento (RCA + RX wireless)

Versione documento: 2.1 · Dual-ESP32 + RS-485 CQ260D + Ingressi RCA + Modulo RX

1. Percorso audio Master – Ingressi RCA → Pad universale → ES8388 → Uscita delay

  Console DJ / sorgente linea (RCA)
        │  (MASTER 2, BOOTH, ecc.)
        ▼
  Ingressi RCA L/R sul box Master
        │
        ▼
  Pad universale per canale (protezione fino a +20/+21 dBu)

      R1 = 15 kΩ serie dal centrale RCA al nodo IN_CODEC
      R2 = 4.7 kΩ dal nodo IN_CODEC a GND analogico

  Impedenza vista dalla sorgente: ~19.7 kΩ
  Attenuazione: ≈ -12.4 dB

        │
        ▼
  ES8388 ADC (line-in stereo)
   - Sample rate rilevato automaticamente (44.1/48 kHz)
   - Input gain regolabile via software (PGA)

        │
        ├─► Elaborazione DSP + audio-reactive
        ├─► TX wireless (ESP-NOW 2 canali)
        └─► Buffer delay (software) → ES8388 DAC → RCA Out L/R

  Uscita RCA L/R:
   - Verso ampli senza DSP, con delay regolabile
   - Livello controllato via gain DAC ES8388 (software)

Note: il pad fisso protegge l'ADC da console molto “calde” (Prime 4, XDJ-XZ). Fine-tuning del livello tramite gain digitale ES8388.

5. Alimentazione – Pololu #5398 (da datasheet)

  Jack+ ──► VIN1 o VIN1R     RJ45 V+ ──► VIN2 o VIN2R     GND comune
  VOUT ──► Buck 5V ≥3A ──► UEDX 5V (SPEC: 5V 1A consigliati)

VINxR = ingresso con ~20 mΩ bilanciamento. Uscita OR: tensione più alta prevale.

7. Bus RS-485: Master → MAX485 → Moduli DSP

  ESP32-S3 Master                         MAX485              Bus RS-485 (115200 8N1)
  ┌──────────────┐                   ┌────────────┐
  │ GPIO43 TX ───┼──────────────────►│ DI         │
  │ GPIO44 RX ◄──┼───────────────────│ RO         │
  │ GPIO10 DE ───┼──────────────────►│ DE + /RE   │
  │ GND ─────────┼──────────────────►│ GND        │
  │              │                   │ A ─────────┼──► A ──► CQ260D #1 ──► #2 ──► ...
  │              │                   │ B ─────────┼──► B ──► (twisted pair)
  └──────────────┘                   └────────────┘

Note: Terminazione 120 Ω tra A e B solo in fondo linea. GND comune tra Master e primo modulo DSP.

8. ES8388 → RCA monitor / delay

  ES8388 DAC LOUT1 / ROUT1 → resistenze serie + filtro RC leggero → centro RCA
  AGND → schermo RCA (punto stella analogico)

  Gain DAC regolato via registri ES8388:
   - outputGainDb configurabile via API/controller
   - permette di adattarsi a ingressi “sensibili” di ampli senza DSP

9. Modulo RX wireless – Audio (XLR + RCA) + DSP + DMX

  MASTER (ESP32-S3)            Wireless (ESP-NOW)               MODULO RX (ESP32-S3)

  ES8388 ADC ──────────┐
                        ├─► TX audio L/R (frame 48 kHz, 16 bit)
  RS-485 DSP (UART2) ───┤─────────────────────────────────────► ESP32 RX → MAX485 locale → CQ260D
  DMX buffer ───────────┘─────────────────────────────────────► ESP32 RX → MAX485 DMX → XLR 3 pin

                                                             ES8388 DAC
                                                             │
                      per canale (L/R)                       │
                      LOUT/ROUT ── 47 Ω ── 4.7–10 µF ──●─────┤
                                                      │
                                                      ├─► IN_L / IN_R modulo DRV134 dual
                                                      │
                                                      └─► RCA_L / RCA_R (uscite linea sbilanciate)

                     DRV134 dual:
                       OUT_L+ / OUT_L− / GND → XLR-L pin 2 / 3 / 1
                       OUT_R+ / OUT_R− / GND → XLR-R pin 2 / 3 / 1

  - Gain ingresso ADC lato Master: gestito via profili sorgente (Denon, Prime 4, XDJ, linea pro/consumer)
  - Gain uscita DAC lato RX: gestito via parametro software (outputGainDb) per adattare il livello all'amplificatore e all'ingresso XLR/RCA locale
