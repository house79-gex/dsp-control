DSP Control – Schemi di collegamento

Versione documento: 2.0 · Dual-ESP32 + RS-485 CQ260D

1. RS-485: Master → MAX485 → Moduli DSP

  ESP32-S3 Master                         MAX485              Bus RS-485 (115200 8N1)
  ┌──────────────┐                   ┌────────────┐
  │ GPIO43 TX ───┼──────────────────►│ DI         │
  │ GPIO44 RX ◄──┼───────────────────│ RO         │
  │ GPIO10 DE ───┼──────────────────►│ DE + /RE   │
  │ GND ─────────┼──────────────────►│ GND        │
  │              │                   │ A ─────────┼──► A ──► CQ260D #1 ──► #2 ──► ...
  │              │                   │ B ─────────┼──► B ──► (twisted pair)
  └──────────────┘                   └────────────┘

Note: Terminazione 120 Ω tra A e B solo in fondo linea. DE=/RE alto durante TX.

5. Alimentazione – Pololu #5398 (da datasheet)

  Jack+ ──► VIN1 o VIN1R     RJ45 V+ ──► VIN2 o VIN2R     GND comune
  VOUT ──► Buck 5V ≥3A ──► UEDX 5V (SPEC: 5V 1A consigliati)

VINxR = ingresso con ~20 mΩ bilanciamento. Uscita OR: tensione più alta prevale.

6. RJ45 lato box – bus DSP (esempio pinout)

PinSegnale
1–2GND
4RS-485 A
5RS-485 B

Verificare con datasheet moduli PDA1001.

7. XLR bilanciato → ES8388 (THAT1206 per canale)

  XLR2 (+) → IN+    XLR3 (−) → IN−    XLR1 → GND
  THAT1206 OUT → C 10µF → ES8388 LIN / RIN (linea sbilanciata)

8. ES8388 → RCA monitor

  LOUT1 / ROUT1 → centro RCA | AGND → schermo RCA

9. Relay DPDT + TCA9535

  TCA9535 P0_0 → base NPN → relay coil (flyback diode)
  LOW = Mixer pass-through | HIGH = Test tone (DAC)

10. Due MAX485

MAX485 #1 (Master): solo bus DSP 115200 → RJ45.

MAX485 #2 (Slave): solo DMX 250k → XLR 3 pin.
